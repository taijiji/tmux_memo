# tmux_memo
tmux memo

# Default Commands

start tmux

```
tmux
```

split vertical pain

```
Ctr + b -> %
```

split hirizontal pain

```
Ctr + b -> "
```

move to next pain

```
Ctr + b -> o
```

move to a numbered pain

```
Ctr + b -> q -> <number>  
```

remove a pain

```
Ctr + b -> x
or
exit
```
